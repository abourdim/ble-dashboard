BLE Dashboard — Start Here

Your guide to getting up and running in 5 minutes

What You Need

💻
Computer
Linux, macOS, Raspberry Pi, or Windows
🐍
Python 3.10+
Backend server runs on Python
📶
Bluetooth Adapter
Built-in or USB BLE 4.0+ dongle
🌐
Chrome or Edge
Web Bluetooth requires these browsers
Optional: A HackRF One SDR dongle for real spectrum analysis. Without it, use the built-in Demo Mode!

Installation — Step by Step

1

Download the project

Clone or download the BLE Dashboard folder to your computer.

git clone https://github.com/your-repo/ble-dashboard.git
cd ble-dashboard
2

Run the launcher

The launcher is an interactive menu that handles everything for you.

bash launch.sh

This works on Linux, macOS, and Raspberry Pi. It will detect your system automatically.

3

Install dependencies

From the launcher menu, press 2 (Install / update dependencies).

This creates a Python virtual environment and installs FastAPI, bleak, and other packages.

On Linux/Raspberry Pi, you may also need: sudo apt install bluetooth bluez
4

Start the server

From the launcher menu, press 3 (Start server).

The server starts on http://localhost:8000 by default.

# Or start manually:
cd backend
pip install -r requirements.txt
python main.py
5

Open the dashboard

Press 5 (Open web dashboard) or open http://localhost:8000 in Chrome or Edge.

Important: Safari and Firefox do NOT support Web Bluetooth. Use Chrome or Edge.

Your First BLE Scan

1

Click "Scan"

The scanner searches for nearby Bluetooth devices. You'll see them appear in the device list with signal strength bars.

2

Click "Connect" on a device

The Characteristics panel opens automatically. Click "Discover" to see all services and characteristics.

3

Read, Write, Subscribe

Each characteristic shows badges for what you can do: R (read), W (write), N (notify/subscribe).

Click Subscribe to watch live data stream into the chart!

No BLE device handy? Click the 🎲 button for demo mode — it simulates 6 devices so you can explore the interface.
